파일을 읽는 중입니다…

데이터

파일을 추가하면 매출이 합산됩니다. 이 영역에 끌어다 놓아도 추가됩니다. 데이터는 초기화 전까지 브라우저에 저장되어, 닫았다 다시 열어도 유지됩니다.
📦 실재고(월말 실사재고) 엑셀(품번·규격·실재고수량 열)을 올리면 안전재고의 부족수량, 폐기검토의 악성재고 수량 계산에 반영됩니다. 표에서 직접 입력·수정도 가능합니다.
📊 BEP(손익분기점) ↔ 연·월 중 한쪽에 입력하면 다른 쪽이 자동 계산됩니다(월=연÷12). 그래프에 보라색 BEP 라인과 남은 월 필요액이 표시됩니다.
👥 사업부 인원 (월별, 1인 매출 반영) 월별 인원을 입력하면 1인당 월매출/목표에 즉시 반영됩니다.
전년 1월부터 올해 12월까지 월별 인원입니다(기본 53명 표시). 값을 바꾸면 1인당 월매출/목표에 즉시 반영·저장됩니다.
☁️ 구글 동기화 (전체 설정을 공용 구글시트와 공유 — 모두에게 공유)
🔄 동기화 기준 (언제 동기화되나요?) — 펼치기/접기
• 파일을 처음 열 때 (○ 자동) — 시작 시 구글시트 URL·자동 옵션이 설정돼 있으면 서버 데이터를 자동으로 불러와 기준점으로 잡습니다.
• 필드 수정 시 (○ 자동, 약 5초 후) — 담당자·신용등급·메모·목표% 등을 수정하면 5초 뒤에 바뀐 항목만(델타) 업로드됩니다. 🔄 버튼에 "업로드 5→1초" 카운트가 보입니다. 다른 사람이 다른 칸을 고쳐도 서로 덮어쓰지 않습니다.
• 메뉴 클릭 시 (✕ 안 됨) — 분석 메뉴를 누르는 것만으로는 동기화되지 않습니다.
• 수동 버튼 — 🔄 동기화/⬆ 저장은 즉시 업로드+전체 받기, ⬇ 불러오기는 받기만 합니다.
• 백그라운드 (○ 60초) — 60초마다 자동으로 서버 변경을 받아옵니다(편집 중이거나 화면이 꺼져 있으면 건너뜀).
URL·비밀키 입력 시 자동 저장됩니다. 불러오기=시트의 전체 설정 가져오기, 저장=현재 전체 설정을 시트에 병합. 비밀키를 아는 사람만 접근됩니다.
