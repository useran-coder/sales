데이터

파일을 추가하면 매출이 합산됩니다. 이 영역에 끌어다 놓아도 추가됩니다. 데이터는 초기화 전까지 브라우저에 저장되어, 닫았다 다시 열어도 유지됩니다.
📊 BEP(손익분기점) ↔ 연·월 중 한쪽에 입력하면 다른 쪽이 자동 계산됩니다(월=연÷12). 그래프에 보라색 BEP 라인과 남은 월 필요액이 표시됩니다.
👥 사업부 인원 (월별, 1인 매출 반영) 월별 인원을 입력하면 1인당 월매출/목표에 즉시 반영됩니다.
전년 1월부터 올해 12월까지 월별 인원입니다(기본 53명 표시). 값을 바꾸면 1인당 월매출/목표에 즉시 반영·저장됩니다.
☁️ 구글 동기화 (전체 설정을 공용 구글시트와 공유 — 모두에게 공유)
URL·비밀키 입력 시 자동 저장됩니다. 불러오기=시트의 전체 설정 가져오기, 저장=현재 전체 설정을 시트에 병합. 비밀키를 아는 사람만 접근됩니다.
